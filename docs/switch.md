
### Props 参数说明

| 属性     | 类型| 参数说明  | 参数   |  默认值  |
| -------- | -----:| -----:  |-----:  | :----:  |
| text     |  String|开关文字描述|   |       |
| width     |  Number|开关宽度|   |        |
| icon    |String | 开关图标描述   |     |
| value     |  Boolean|开关状态|true，false   | true       |
| disabled     |  Boolean|开关是否禁用|true，false   | false       |
| size |  String  |  开关尺寸 | large(大) / mini(迷你)	 | medium 
| onValue |  String,Number,Boolean  |  自定义开值 | true	 | 
| offValue |  String,Number,Boolean  |  自定义关值 | false	 |
