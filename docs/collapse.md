
### Props 参数说明

| 属性     | 类型| 参数说明  | 参数   |  默认值  |
| -------- | -----:| -----:  |-----:  | :----:  |
| value/v-model | String / Array | 当前激活的面板(如果是手风琴模式，绑定值类型需要为string，否则为array) |  - |   -  |
| accordion | Boolean | 是否是手风琴模式 |  true / false |  false  |

### Events 事件说明

| 属性     | 类型| 说明  | 回调参数   | 
| -------- | -----:| -----:  |-----:  | 
| change |  Events  | 选中值发生变化时触发 |  目前的选中值, 是否展开 |


### Item 参数说明

| 属性     | 类型| 参数说明  | 参数   |  默认值  |
| -------- | -----:| -----:  |-----:  | :----:  |
| title | String | 面板标题 | - | - |
| code |   String / Number | 唯一标识符 | - | - |
